Прошивка ESP32-S3 через браузер

⚡ Прошивка

Браузер не поддерживается. Используйте Chrome, Edge или Opera. Требуется HTTPS
⚠️ Подключите ESP32-S3 по USB. Если не определяется — зажмите BOOT при подключении.

📋 Инструкция

  1. Подключите ESP32-S3 к USB
  2. Нажмите "Установить прошивку"
  3. Выберите COM-порт
  4. Дождитесь завершения (~1 мин)
  5. Настройте WiFi через веб-интерфейс
